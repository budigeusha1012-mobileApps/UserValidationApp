User Validation App
A simple Android application built using Kotlin and XML with Clean MVVM Architecture.

Features
User Form Validation
Real-time Error Messages
Submit Button Enable/Disable
Retrofit API Integration
MockAPI POST Request
Success Screen with User Details
Loading Indicator
Error Handling
Navigation Component
Tech Stack
Kotlin
XML
MVVM Architecture
Retrofit
Coroutines
LiveData
Navigation Component
ViewBinding
Project Structure

Validation Rules
Name must contain minimum 2 characters
Email must be valid
Phone number must contain 7–15 digits
City must be selected
